Getting a Link Preview of a URL

To retrieve a link preview, simply send a Get Request with your apikey tag in the header and the url as an query parameter

Curl GET Request example

curl --verbose --header "apikey: ddaf02d1-11cd-49c0-a4e6-3a13976315d3"
--request "GET" "https://api.teenielink.dev/v1/preview?url=https://example.com" {{jsonExample | json}}
